# 入力ポリシーガイド (Safety First)

本アプリケーションを安全かつ効果的に利用するための、入力に関するガイドラインです。

## 1. 入力してよい情報 (推奨)
- **日常生活動作 (ADL) の状態**: 「一人で歩ける」「着替えに介助が必要」など。
- **認知機能・精神状態**: 「最近物忘れが増えた」「夜間に不安を訴える」など。
- **生活環境**: 「段差が多い」「独居である」など。
- **本人の希望・意向**: 「最期まで自宅で過ごしたい」「デイサービスは嫌だ」など。

## 2. 伏せるべき情報 / 入力を控えるべき内容 (禁止)
- **具体的な個人情報**: 氏名、詳細な住所、電話番号、生年月日等は、可能な限り抽象化してください（例：「A様」「新宿区」）。
  - ※内部でマスキング処理を行っていますが、安全のため入力段階でも配慮をお願いします。
- **過度に主観的な悪口・ゴシップ**: 家族関係のトラブル等は、ケアプランに必要な範囲で客観的な事実のみを記載してください。
- **特定の医療診断名への過度な依存**: AIは医療診断を行えません。診断名に基づいた直接的な指示を求める入力は控えてください。

## 3. 推奨入力フォーマット
以下のような構成で入力すると、AIがより正確なケアプラン案を生成できます。
```text
【現在の状況】（ADL、認知機能、環境）
【課題】（本人が困っていること、リスク）
【希望】（本人・家族の目標）
```

## 4. 運用上の注意点
- **生成された情報の確認**: AIが生成した内容はあくまで「下書き」です。必ず専門職の目で内容の妥当性、安全性、倫理性を確認してください。
- **禁止表現のチェック**: アプリケーション側で「絶対に」「不明」などの不適切な表現をチェックしていますが、最終的な文言調整は人間が行ってください。
