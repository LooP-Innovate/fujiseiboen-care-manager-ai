# ケアプランAI連携アプリ RC版（リリース候補版） 実務テストガイド

## 1. RC版の仕様メモ（現状整理）

### 現在できること
- 8項目のフォーム入力に基づき、居宅サービス計画書（第2表）の「草案」となる9項目のテキストを約10〜15秒で生成します。
- 入力されたキーワード（独居、胃ろう等）に反応して、内部的に「支給限度額」「医療的チェックポイント」「インフォーマル資源リスト」の静的知識を自動で読み込み、制度・安全要件に即した提案を行います。
- 退院サマリー等のテキスト（100KB以内）を読み込み、主入力との矛盾やアセスメント不足情報を自動的に検知・指摘します。
- 生成結果は見出しごとに整理され、ワンクリックでクリップボードにコピーできます。

### まだできないこと（次期開発候補）
- PDFファイルやWordファイルなどをそのまま直接読み込むこと（コピー＆ペースト、または.txtファイルの読込は可能）
- 過去の入力履歴の保存機能（ブラウザ画面を更新すると入力内容が消えます）
- 全ての市区町村ローカルルールの完璧な反映
- マルチモデル推論（内部でOpenAI接続機能は保持していますが、現行UIではGeminiの単独高速処理に固定しています）

### 入力項目一覧
1. 要介護度（必須）
2. 本人・家族の希望（意向）（必須）
3. 主疾患・医療的ケア・服薬状況（必須）
4. ADL・身体状況（必須）
5. 認知機能・精神状況（必須）
6. 家族構成・支援体制（必須）
7. リスク・留意点（必須）
8. 氏名・年齢・性別（基本属性）
9. 補助情報・参考資料（任意：長文テキスト読込欄）

### ⚠️ 注意点（大原則）
**AIの出力はあくまで「人間が編集するための草案作成補助」です。**
AIが提案するサービス内容や目標は、事実誤認や過剰提案を含む可能性があります。必ずケアマネジャーご自身の目で事実関係を確認し、必要に応じて加筆・修正した上で業務に利用してください。

## 2. 現場テスト（実務検証）ガイド

### 1分で分かる使い方
1. **入力する**: 左側のフォームに、対象者のアセスメント情報を入力してください。長文ではなく、箇条書きや単語の羅列でも十分にAIが文脈を補完します。
2. **資料を取り込む（任意）**: お手元に退院サマリーや引継ぎメモのテキストがあれば、一番下の「ファイル読込」ボタンから読み込むか、空欄に直接貼り付けてください。
3. **生成する**: 「✨ ケアプラン草案を生成」ボタンを押して十数秒お待ち下さい。
4. **確認・コピーする**: 右側に結果がカード形式で表示されます。「📝 草案をコピー」ボタンを押し、お使いの介護ソフト等に貼り付けて修正してください。

### 入力時のコツ
- **本音と建前を分ける**: 本人と家族で意見が違う場合、「【本人】家を出たくない 【家族】転倒が心配で施設に入れたい」と別々に書くと、AIが両方に配慮した目標や着地点の草案を作りやすくなります。
- **リスクは具体的に**: 単に「転倒リスクあり」だけでなく、「玄関の段差」「深夜の頻尿」など環境的・身体的要因を少し足すだけで、住宅改修や適切な福祉用具の提案精度が飛躍的に上がります。

### 出力の見方・読み方
- **⚠️ 注意点 / 🚨 不足情報**: AIが「このままプランを組むのは危険」「アセスメントが足りない・矛盾している」と判断した箇所です。**実務上、このアラートに最も価値があります。** 必ず目を通し、利用者への追加の聞き取り調査や主治医確認の足がかりにしてください。
- **参考にした観点**: AIがどの保険ルールや内部知識を参考にしてプランを組んだかが記載されます。

## 3. テスト時の確認観点とフィードバックのお願い

ケアマネジャーの皆様には、実際の担当ケース（個人情報を伏せたもの）をいくつか入力し、以下の観点でのフィードバックをお願いいたします。

1. **草案の品質**
   - 課題抽出や目標設定は、プロとして納得できる（あるいは編集の土台になる）レベルか？
   - 提案されるサービスが限度額や生活実態に対して過剰すぎないか？
2. **不足情報の指摘精度**
   - AIが指摘する「不足情報」や「注意点」は、実務上のハッとする「気づき」として役に立つか？
3. **使い勝手**
   - 8項目の入力ボリュームは日常業務の負担にならないか？
   - 右ペインの結果は見やすく、コピーして編集しやすいか？
4. **追加してほしい「独自の知識」**
   - 各地域の特有のルールや、利用の多い配食サービス一覧など、「これもAIの知識に読ませておいてほしい」という資料（テキスト）があれば、随時開発チームへご提供ください。
